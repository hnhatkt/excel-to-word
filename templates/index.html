Excel → Word

Vui lòng chuẩn bị file Excel theo cấu trúc:

- Hàng 1: tiêu đề cột (không dấu, viết liền bằng dấu "_")
Ví dụ: ten_san, dia_chi, so_hop_dong...

- Hàng 2 trở đi: dữ liệu, mỗi dòng tương ứng 1 file Word

- File excel phải có 3 cột: so_hop_dong, ten_kem_theo, ten_san

- Tên file Word sẽ được tạo theo:
so_hop_dong + ten_kem_theo + ten_san.docx

- Trong file Word template sử dụng biến:
{{ ten_san }} {{ so_hop_dong }} {{ ten_kem_theo }}
